About

This is an experimental and educational project by Max Caplan and Will Macdonald throughout the duration of a summer internship.

This project is implementing the use of machine learning through both computer vision and audio authentication for the purpose of developing a multi-step authentication process, with aims of creating a more reliable and secure login system.

Please see our GitHub repository at Multipart-Authentication for more information on the code and licensing.
Pull requests are welcome. We ask that for major changes, an issue is created to outline the details of the changes.
